# Antigravity Workspace

Este repositorio contiene el código fuente para mis proyectos con Antigravity, organizado por proyectos.

## Estructura

- **projects/**: Contiene todos los proyectos individuales.
  - **SAM/**: Asistente de IA para D&D (Backend, Frontend, Modelos).
  - **FF8/**: Proyecto principal de gestión (Inversionistas, RRHH, Comercial).
  - **_template/**: Plantilla para nuevos proyectos.
- **playground/**: Scripts de prueba y experimentación (incluye `main.py`).
- **shared/**: Recursos compartidos.

## Flujo de trabajo

1. Para nuevos proyectos, copiar `projects/_template`.
2. Desarrollar dentro de la carpeta del proyecto correspondiente.
